Lazypic은 콘텐츠 제작 파이프라인을 협업하며 개발합니다.
콘텐츠를 기획하며 콘텐츠 제작 노하우와 파이프라인을 Github에 오픈소스로 공유합니다.
1인 콘텐츠 개발자, 프리랜서그룹, 기업, 비영리단체, 학교등과 기술 협업을 진행하고 있습니다.
Lazypic collaborates and develops content creation pipelines.

We are working on technical collaboration with one-man content developers,

License

코드의 가치보다 사용 가치에 중점을 두고 오픈소스로 개발하고 있습니다.

소프트웨어 협업, 버전관리, 배포, 이슈관리, 유지보수의 편의성을 위해 모든 프로젝트를 오픈소스로 진행합니다.

효율적인 진행을 위해 협업도구로 Github와 Slack을 사용하고 있습니다.

제작되는 소프트웨어는 BSD 라이센스 정책을 따릅니다.

Cloud

우리는 Amazon Web Service를 사용하고 있습니다.

안정적인 인프라, 신뢰할 수 있는 보안체제, 클라우드 서비스내 국제화 서비스를 위한 다양한 개발도구를 활용할 수 있기 때문입니다.

클라우드를 이용해서 협업시 클라우드에 접근할 수 있는 권한 및 키 발급은 admin@lazypic.org 을 통해서 진행됩니다.

Start-Up Support

콘텐츠 제작사 초기, 보안 컨설팅 지원을 할 수 있습니다. 예) MPAA

초기 회사 마케팅에 필요한 도메인 설정, 도메인과 연결된 메일 또는 어플리케이션, 도메인 전용 메일 셋팅을 지원합니다.

개발자가 없는 환경에서 회사 AWS Console 관리, 코드 리포지터리 관리를 지원합니다.

WebApp

무선 인터넷환경, 사내환경에서 사용가능한 WAS(Web Application Server)를 제작하고 있습니다.

아이폰, 아이패드, 노트북, 데스크톱을 동시에 지원하는 웹 어플리케이션을 고려하며 개발합니다.

Web Site

기업 마케팅, 고객에게 필요한 사이트 제작을 진행합니다.

홈페이지와 연동되는 웹 어플리케이션을 제작합니다.

단순한 홈페이지가 아닌 클라이언트와 관계자들이 서로 Co-work을 할 수 있는 APP 형식의 홈페이지를 제작합니다.

Education

리눅스, 클라우드, 프로그래밍 교육을 진행합니다.

교육 문서는 Github에 작성되며, 생성된 문서와 예제 코드는 교육 콘텐츠로 활용됩니다.

회사 콘텐츠 제작 파이프라인 컨설팅을 지원합니다.

Linux, Python, Cloud(AWS), Go 교육프로그램 진행을 진행합니다.

Open-source solution, FFmpeg, OCIO, OIIO, OTIO, OpenVDB, Alembic, USD 등의 오픈소스를 사용하여 관련 개발을 진행합니다.

Development

콘텐츠 제작에 필요한 Windows, Linux, macOS 용 시스템 명령어를 제작합니다.

회사에서 필요한 프로그램 API 작성합니다.(RestAPI, Go, Python을 주 언어로 사용합니다.)

단순 반복적인 작업에 대한 자동화 스크립트를 제작합니다.

소프트웨어를 빌드를 위한 테스트환경은 Github Actions를 사용하고 있습니다.

기본적인 테스트를 통해 기업에서 사용하는 소프트웨어 안정성 향상을 위해 노력하고 있습니다.

Testing

Test build open source solutions for the future.
